# ATS-Friendly Resume Templates

Free, open-source resume templates engineered to pass Applicant Tracking Systems (ATS). Every template is single-column with standard headings and no tables, text boxes, images, or multi-column layouts that ATS parsers mangle.

## Why ATS-friendly formatting matters
Most mid-to-large employers filter resumes through an ATS before a human ever reads them. Visually fancy templates with columns, icons, headers/footers, and graphics are frequently parsed into garbage, so your experience never gets scored. These templates avoid every common parser trap:

- Single column, clean top-to-bottom reading order
- Standard section headings (Experience, Education, Skills)
- No tables, text boxes, images, or columns
- Real selectable text (never text saved as an image)
- Standard, widely-supported fonts

## How to use
1. Pick a format and copy the file.
2. Replace the placeholder content with your own.
3. Export to PDF. For the HTML template, open it in a browser and choose Print then Save as PDF.
4. Run the result through an ATS checker before you apply.
